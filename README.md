# Prezi to PDF

## What is it

Prezi to PDF is a browser extension that allows users to convert Prezi presentations into a single downloadable PDF. The extension iterates through all slides in a presentation, captures screenshots of each slide, and combines them in order. It works directly in the browser and does not collect personal or sensitive data.

## Features

- Automatically captures all slides from a Prezi presentation.
- Generates a single PDF with slides in sequence.
- Compatible with both Firefox and Chromium-based browsers.
- Lightweight and easy to use.
- Requires only standard permissions: tabs, downloads, and storage.

## Installation

### Firefox

1. Download the latest `.zip` file from the [Releases]([https://github.com/Dewnan/Prezi2PDF/releases](https://github.com/Dewnan/Prezi2PDF/releases/tag/v1.0.2)) page.  
2. Extract the zip file.  
3. Open `about:debugging` in Firefox.  
4. Click "This Firefox" on the left sidebar.  
5. Click "Load Temporary Add-on".  
6. Select the `manifest.json` file from the extracted folder.  
7. The extension will now appear in your toolbar and is ready to use.

### Chromium (Chrome, Edge, etc.)

1. Download the latest `.zip` file from the [Releases]([https://github.com/Dewnan/Prezi2PDF/releases](https://github.com/Dewnan/Prezi2PDF/releases/tag/v1.0.2)) page.  
2. Extract the zip file.  
3. Open `chrome://extensions` in your browser.  
4. Enable "Developer Mode" in the top right corner.  
5. Click "Load unpacked".  
6. Select the extracted folder containing `manifest.json`.  
7. The extension will now appear in your toolbar and is ready to use.

## Usage

1. Open a Prezi presentation in your browser.  
2. Click the Prezi to PDF extension icon.  
3. The extension will capture each slide and generate a PDF for download.
